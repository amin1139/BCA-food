# 🍔 BCA Food Wala

**BCA Food Wala** is a modern and responsive food ordering web application built using **React JS**.  
The app fetches live restaurant and menu data using the **Swiggy Live API** and provides a smooth user experience with optimized performance.

---

## 🚀 Features

- 🔴 Live restaurant listing from Swiggy API  
- 🍽️ Dynamic restaurant menu page  
- 🛒 Add to cart functionality using Redux Toolkit  
- ⚡ Lazy loading for faster performance  
- 🧭 Seamless navigation with React Router  
- 🎨 Responsive UI built with Tailwind CSS  
- 🌐 Online/Offline status detection  
- ✨ Clean and scalable component architecture  

---

## 🛠️ Tech Stack

- **Frontend:** React JS  
- **State Management:** Redux Toolkit  
- **Routing:** React Router  
- **Styling:** Tailwind CSS  
- **Performance Optimization:** React Lazy & Suspense  
- **API:** Swiggy Live API
